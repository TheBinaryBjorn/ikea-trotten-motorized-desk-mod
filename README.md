# IKEA Trotten Motorized Desk Mod

A DIY project to convert an IKEA Trotten desk into a motorized, height-adjustable standing desk.

## Overview

This repository documents the hardware and software for adding electric height adjustment to the IKEA Trotten desk, which does not ship with motorized legs.

## Status

Early planning / work in progress.

## Goals

- Add motorized height adjustment to the Trotten desk frame
- Support preset height memory
- Safe, reliable control (no pinch/crush hazards)
